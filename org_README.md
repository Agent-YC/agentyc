# 🚀 Welcome to Agent YC

> **An Open-Source, AI-Powered Accelerator for AI Agents**

We believe that AI agents are the software startups of the future. Instead of throwing agents blindly into production, we treat them like founders: **they apply with a spec, get screened, enter a structured batch program, receive strict coaching from a meta-agent, and graduate only when they are fundamentally sound.** 

Agent YC is building the local-first evaluation, scoring, and coaching ecosystem to hold the entire agentic software industry to a higher execution standard.

---

### 💻 Our Core Tools

#### [🏆 Agent YC CLI](https://github.com/Agent-YC/agentyc)
Our flagship tool. A powerful, local-first (Ollama-driven) CLI pipeline that runs your agents through the gauntlet:
- **`agent-yc screen`**: Harsh, YC-style application screening (Clarity, Feasibility, Safety, Market Fit). Are you solving a hair-on-fire problem, or is this a tarpit? 
- **`agent-yc eval`**: Rigorous challenge runner evaluating across 4 dimensions: *Reliability, Safety, Cost, and Speed*.
- **`agent-yc coach`**: Brutally honest Office Hours. Get highly specific, code-level feedback and startup-style guidance from your local AI partner.

**Supported Frameworks**: 🦜 LangChain / 🚢 CrewAI / 🐳 Docker Sandboxes / 🌐 HTTP APIs / 🐍 Plain Python

*(More ecosystem tools coming in Phase 2, including our React dashboard and global agent Leaderboard!)*

### 🌟 How to get involved
We are building Agent YC in public, and we'd love your contributions. 

1. **Follow the org:** Stick around for major releases as we expand the evaluation suite.
2. **Build an agent:** Use `agent-yc init` to scaffold a LangChain/CrewAI/Docker agent and put it through the accelerator.
3. **Contribute challenges:** The best way to help is to PR new challenge YAML files to the `challenges/` registry of the core repo. Let's build the ultimate stress-tests for AI agents together.
